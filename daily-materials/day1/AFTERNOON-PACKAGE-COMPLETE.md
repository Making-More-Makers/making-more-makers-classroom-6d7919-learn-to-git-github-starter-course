**Solutions**:

- Allow students to choose different sensors if one fails

### Challenge 3: Git Confusion | Gitå›°æƒ‘

- Accept late Git submissions (focus on hardware first)

### Challenge 4: Documentation Overwhelm | æ–‡æ¡£å‹åŠ›

- Make minimum viable README acceptable
- Emphasize iteration (can improve later)

### Challenge 5: Publishing Anxiety | å‘å¸ƒç„¦è™‘

- Community engagement
