## ğŸ“ Requirements

### Minimum Requirements

Your project MUST include:

1. **At least 1 input device**:
   - Button
   - Potentiometer
   - Any sensor from lab

2. **At least 1 output device**:
   - LED
   - Buzzer
   - Servo motor
   - RGB LED

3. **Meaningful interaction**:
   - Input affects output in a logical way
   - Not just "button turns on LED"
   - Show creativity!

4. **Working code**:
   - Compiles without errors
   - Runs as intended
   - Well-commented

5. **Complete documentation**:
   - README with project description
   - Circuit diagram or clear photo
   - Instructions for use

## ğŸ’¡ Project Ideas

Need inspiration? Try one of these:

### Beginner Level
- **LED Dimmer**: Potentiometer controls LED brightness
- **Reaction Game**: LED lights up, press button quickly, measure time
- **Night Light**: Light sensor turns on LED when dark

### Intermediate Level
- **Servo Angle Control**: Potentiometer controls servo position
- **Button Counter**: Button increments counter, display via serial or LEDs
- **Mood Indicator**: Different LEDs light based on potentiometer position

### Advanced Level
- **Simple Alarm**: Sensor triggers buzzer and LED
- **Interactive Pattern**: Button changes LED blinking pattern
- **Multi-Mode System**: Button cycles through different behaviors
